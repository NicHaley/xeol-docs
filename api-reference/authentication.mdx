---
title: "API Key"
description: "API Key for Xeol's EOL API"
---

In order to use Xeol's EOL API in production, you will need an `API Key` and pass it in the header with all your requests.

### Generate API Key
Message `founders@xeol.io` to get an `API Key`.

### Use API Key
```bash
"Authorization": "<XEOL_API_KEY>"
```
